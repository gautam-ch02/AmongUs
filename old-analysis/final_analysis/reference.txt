8. DATA LIMITATIONS
================================================================================

  - 32/70 games ended before any meeting (0 SPEAK events). These games have
    empty v2 deception files — this is data-inherent, not a pipeline error.
  - 1 game (C07/2026-02-27_exp_3) has 18 SPEAK events but all are "..."
    (Llama 3.1 8B generated only ellipsis). Produces 0 extractable claims.
  - C05 and C06 have only 5 games each. Statistical comparisons involving
    these configs have low power.
  - v2 claim coverage averages 73.6% across games with SPEAK events
    (up from 32.2% in v1). Some SPEAK events contain no extractable claims.
  - All crewmate factual lie rates are 0.0% (verified correct — crewmates
    have no ground-truth motivation to lie about observable facts).
